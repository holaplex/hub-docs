A blockchain wallet is a digital wallet that allows users to securely store, manage, and transfer their cryptocurrencies or other digital assets on a blockchain network.

```graphql
type Wallet {
  address: String
  mints: [CollectionMint!]
  treasuryId: UUID!
  createdAt: DateTime!
  removedAt: DateTime
  createdBy: UUID!
  assetId: AssetType!
  id: UUID!
  deductionId: UUID
}
```
