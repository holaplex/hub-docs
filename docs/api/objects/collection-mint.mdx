Represents a single NFT minted from a collection.

```graphql
type CollectionMint {
  id: UUID!
  collectionId: UUID!
  address: String
  owner: String!
  creationStatus: CreationStatus!
  createdBy: UUID!
  createdAt: DateTime!
  signature: String
  edition: Int!
  sellerFeeBasisPoints: Int!
  creditsDeductionId: UUID
  compressed: Boolean!
  collection: Collection
  metadataJson: MetadataJson
}
```
